# 📊 Sales Report Dashboard — Power BI

> Projeto desenvolvido como parte da **Formação Power BI Analyst** da [DIO](https://www.dio.me/)

---

## 📌 Visão Geral

Este projeto consiste em um **Dashboard de Vendas e Lucros** construído no Power BI. O relatório foi estruturado em três páginas principais: **Sales**, **Profit** e **Report**, cada uma com foco em perspectivas distintas de análise do desempenho comercial.


---

## 🛠️ Tecnologias Utilizadas

| Ferramenta | Uso |
|---|---|
| **Power BI Desktop** | Criação do dashboard e modelagem de dados |
| **Excel / CSV** | Fonte de dados financeiros |
| **DAX** | Medidas calculadas (KPIs, totais, variações) |
| **Power Query** | Transformação e limpeza dos dados |

---

## 📈 Principais Insights

- O segmento **Government** é o maior em volume de vendas (53 Mi), seguido de **Small Business** (42 Mi).
- O **4º Trimestre** de 2014 registrou o maior faturamento: **29,75 Mi**, impulsionado pelo mês de **outubro**.
- O lucro cresceu significativamente de 2013 para 2014: de **3,87 Mi** para **13,01 Mi** (+235%).
- O mês de **outubro** concentra o maior pico de vendas brutas (Gross Sales: 22 Mi).
